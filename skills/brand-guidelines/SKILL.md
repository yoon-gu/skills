# Anthropic 브랜드 스타일링

## 개요

Anthropic의 공식 브랜드 아이덴티티와 스타일 리소스에 접근하려면 이 스킬을 사용하세요.

**키워드**: 브랜딩, 기업 아이덴티티, 시각적 아이덴티티, 후처리, 스타일링, 브랜드 색상, 타이포그래피, Anthropic 브랜드, 시각적 서식, 시각적 디자인

## 브랜드 가이드라인

### 색상

**주요 색상:**

- 다크: `#141413` - 주요 텍스트 및 어두운 배경
- 라이트: `#faf9f5` - 밝은 배경 및 어두운 배경 위 텍스트
- 미드 그레이: `#b0aea5` - 보조 요소
- 라이트 그레이: `#e8e6dc` - 은은한 배경

**강조 색상:**

- 오렌지: `#d97757` - 주요 강조색
- 블루: `#6a9bcc` - 보조 강조색
- 그린: `#788c5d` - 3차 강조색

### 타이포그래피

- **제목**: Poppins (Arial 대체)
- **본문**: Lora (Georgia 대체)
- **참고**: 최상의 결과를 위해 환경에 폰트가 사전 설치되어 있어야 합니다

## 기능

### 스마트 폰트 적용

- 제목(24pt 이상)에 Poppins 폰트 적용
- 본문에 Lora 폰트 적용
- 커스텀 폰트를 사용할 수 없는 경우 자동으로 Arial/Georgia로 대체
- 모든 시스템에서 가독성 유지

### 텍스트 스타일링

- 제목(24pt+): Poppins 폰트
- 본문: Lora 폰트
- 배경에 따른 스마트 색상 선택
- 텍스트 계층 구조 및 서식 유지

### 도형 및 강조 색상

- 텍스트가 아닌 도형에 강조 색상 사용
- 오렌지, 블루, 그린 강조색을 순환 적용
- 브랜드에 맞으면서 시각적 흥미 유지

## 기술 세부사항

### 폰트 관리

- 사용 가능한 경우 시스템에 설치된 Poppins 및 Lora 폰트 사용
- Arial(제목) 및 Georgia(본문)로 자동 대체 제공
- 폰트 설치 불필요 - 기존 시스템 폰트로 작동
- 최상의 결과를 위해 환경에 Poppins 및 Lora 폰트를 사전 설치하세요

### 색상 적용

- 정확한 브랜드 매칭을 위해 RGB 색상 값 사용
- python-pptx의 RGBColor 클래스를 통해 적용
- 다양한 시스템에서 색상 충실도 유지
